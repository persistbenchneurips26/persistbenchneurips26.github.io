Can 4D Foundation Models Remember?

NeurIPS 2026 Submission

PersistBench evaluates object-centric memory with 360° video as omniscient ground truth, probing whether 4D foundation models truly remember objects even after leaving the input view.

View Leaderboard View Visualization
Overview

Reference-based evaluation for the hidden state of the world.

Perceiving and remembering the visual world is fundamental for navigating dynamic environments. Current 4D foundation models can reconstruct and generate scenes, but can they remember what they have seen?

Drawback

Incomplete memory evaluation

01

Visible-view bias

Common benchmarks reward rendering what is still observable, conflating novel-view synthesis with memory.

02

No hidden reference

Standard videos lose ground truth once objects exit the frame, so off-screen object state cannot be checked.

03

Reference-free scoring

Plausibility and consistency scores can miss whether the same observed entity was faithfully retained.

Solution

Object-centric visual memory evaluation

01

Invisible-view memory

Evaluates objects after they leave the input view, separating memory from visible-view synthesis.

02

Ground-truth hidden state

Uses real-world 360° observations to supervise object state beyond the perspective crop.

03

Reference-based scoring

Checks whether the same observed entity is preserved in existence, motion, and appearance.

Memory Aspect

Visual memory is not a single number.

Inspired by human visual memory, PersistBench decomposes remembering into object permanence, spatiotemporal continuity, and appearance-based identity preservation.

Dataset

The answer is 360° video.

PersistBench starts with an omniscient 360° recording. From the same timestamp sequence, we crop one view as the model input and another view as the held-out reference. The reference crop and object mask let us test whether the model remembers the target after it leaves the input shot.

Findings

What PersistBench reveals.

01

Seeing is not remembering

Every evaluated model drops once the target object becomes invisible, separating novel-view synthesis from persistent memory. The average dynamic visible-to-invisible gap is over 35 points across public models.

02

Memory is multi-faceted

Permanence and appearance are tightly coupled, while motion continuity behaves more independently and should be evaluated separately. The analysis reports strong Perm-App correlation and weaker correlations involving Cont.

03

Static and dynamic memory correlate

Models that remember static objects tend to remember dynamic objects, suggesting shared underlying capabilities rather than two unrelated skills.

04

Generative priors help

Generative models often outperform deterministic reconstruction models, especially on object permanence and appearance, because large-scale video priors can fill in plausible unseen regions.

05

Camera diversity matters

Among 360° generators, more diverse camera sampling is linked with stronger memory once objects leave the frame.

06

Explicit conditioning helps

Models that project input observations into target views before inpainting achieve strong invisible-segment scores, indicating that geometric structure can support memory under occlusion.
